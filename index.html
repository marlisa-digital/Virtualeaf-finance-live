Hubungkan ke Google Sheets

Cara Setup (5 menit)

1
Buka Google Sheets baru → Extensions → Apps Script
2
Hapus semua kode yang ada, paste kode dari file virtualeaf-apps-script.js
3
Klik Deploy → New Deployment → Web App
Execute as: Me · Who has access: Anyone
4
Copy URL deployment yang muncul, paste di bawah ini
💡 Tips token: Di Apps Script tambahkan: var SECRET = "token-kamu"; if(params.token !== SECRET) return error("Unauthorized");
Tips Sharing untuk Auditor
Setelah terhubung, Google Sheet kamu otomatis jadi audit trail.
Share ke auditor: buka Google Sheet → Share → View only
